Types of CSS

  1. Inline CSS

    2. Inline CSS is a Css type that is used in html elements by using style attribute

  2. Internel CSS

    3. Internel CSS is a Css type that is used by style tag in head tag of html

  3. Externel CSS

    4. Externel CSS is a Css type that is used by creating seperate file and it needs to link with html file (inking done in head tag of html)
